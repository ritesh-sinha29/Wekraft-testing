# Create Tickets

Learn how to log support tickets directly from the real-time chat composer in WeKraft Teamspace.

---

## Where to Go & How to Go

### Creating a Chat Ticket
- **Where to Go**: Navigate to **Teamspace** in the sidebar and click on any chat channel.
- **How to Go**:
  1. Position your cursor in the **Message Composer** input box.
  2. Type the `/` (Slash) character as the very first character in the composer.
  3. The **Create Ticket** popover dialog will instantly appear.
  4. Configure your ticket:
     - **Ticket Description**: Provide a detailed description of the issue or request (e.g., "Reset password button is throwing a 500 error").
     - **Assignee**: Choose a team member from the dropdown menu list of project collaborators.
  5. Click **Create Ticket** (or press <kbd>Enter</kbd>).
  6. The ticket is immediately inserted into the database and is visible to the team under the channel's Tickets dropdown.

---

## Ticket Fields

| Field | Type | Required | Description |
|---|---|---|---|
| **Ticket Description** | Text area | Yes | Detailed summary of the support request or task. |
| **Assignee** | Dropdown list | Yes | The team member delegated to resolve the ticket. |
