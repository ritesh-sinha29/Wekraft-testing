# Community Hub

The **Community Hub** is Wekraft's discovery space, designed to encourage collaboration and project sharing across different user spaces. It allows developers and managers to find public projects, view open repositories, and request membership to collaborative workspaces.

---

## Capabilities & Roadmap

The Community Hub is currently flagged with a **Soon** badge, indicating features in active development:
- **Public Project Listings**: Browse through projects shared publicly by other users.
- **Discover Projects**: Filter by tags, technologies, and active sprint topics.
- **Join Requests**: Send direct requests to project owners to join their collaborative teams.
