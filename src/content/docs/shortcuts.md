# Keyboard Shortcuts

Master Wekraft and speed up your workflow with these essential keyboard shortcuts. 

## General Navigation

| Shortcut | Action |
|---|---|
| `Cmd + K` | Open Command Palette / Global Search |
| `B` | Toggle Sidebar visibility |
| `Esc` | Close any open Dialog, Sheet, or Modal |
| `?` | Open this Keyboard Shortcuts reference |

---

## Task Management

These shortcuts work whenever you are inside a Project workspace.

| Shortcut | Action |
|---|---|
| `T` | Create a **New Task** |
| `I` | Create a **New Issue** |
| `S` | Go to **Sprint Planning** |
| `E` | Edit the currently selected item |
| `Enter` | Open detailed view for the selected task |
| `Delete` | (Admin only) Delete the selected task |

---

## AI Assistant (Kaya)

| Shortcut | Action |
|---|---|
| `Cmd + J` | Toggle Floating Kaya AI Chat |
| `Alt + K` | Focus Kaya input field in AI Workspace |

---

## VS Code Extension

While focused in the VS Code sidebar:

| Shortcut | Action |
|---|---|
| `Alt + W + O` | Open current task in Wekraft Web |
| `Alt + W + C` | Mark current task as **Completed** |
| `Alt + W + T` | Start/Stop **Time Tracker** |

---

> [!TIP]
> On Windows and Linux, use `Ctrl` instead of `Cmd` for all global shortcuts.
