## What Each Plan Includes

### Free Plan

The Free plan is perfect for individuals and small teams getting started:

- Create up to **2 projects** and join up to **2 projects**
- Up to **3 members** per project (including the owner)
- **2 GB** of cloud storage for attachments and teamspace messages
- Full access to tasks, issues, sprints, calendar, and team space
- VS Code extension in **read-only mode** (view tasks, but can't update status from IDE)
- Basic codebase map visualization
- Project Structure for connected GitHub repositories
- All core features with no time limit

### Plus Plan

The Plus plan is ideal for growing teams that need more capacity:

- Everything in Free, plus:
- Create up to **10 projects** and join up to **10 projects**
- Up to **6 members** per project
- **15 GB** of cloud storage
- **Advanced analytics charts** in the workspace dashboard (team contribution radar, sprint bar chart, severity heatmap, weekly velocity, member workload, weekly engagement)
- VS Code extension in read-only mode

### Pro Plan

The Pro plan unlocks the full power of Wekraft:

- Everything in Plus, plus:
- Create up to **20 projects** and join up to **20 projects**
- Up to **15 members** per project
- **30 GB** of cloud storage
- **Kaya AI Assistant** — 360 calls per month for AI-powered sprint planning, reports, and insights
- **Full two-way VS Code sync** — update task status, complete tasks, and auto-track time from the IDE
- **AI-powered heatmap insights** — Kaya analyzes workload data and provides natural-language recommendations
- **Activity Flow** — visual timeline of team activity patterns
- **GitHub activity overlay** — enrich heatmaps with commit and PR data
- **Priority support** — faster response times from the Wekraft team

## Usage Limits Explained

### Projects (Owned)
This is the number of projects you can create as an **Owner**. If you reach your limit, you must either upgrade your plan or delete an existing project to create a new one. The limit is displayed on the dashboard metric card (e.g., `2/20`).

### Projects (Joined)
This is the number of projects you can join that are owned by other users. The limit is also shown on the dashboard metric card.

### Members per Project
Wekraft enforces a cap on how many people can be in a single project based on the **owner's plan**. This includes all roles (Owner, Admin, Member, Viewer). If you've reached the limit, the system prevents new invite links from being sent or join requests from being accepted.

### Kaya AI Calls
Kaya AI usage is measured in "calls" — each message sent to Kaya counts as one call. Pro users get **360 calls per month**. If you exceed this limit:
- Previous conversations remain readable
- New queries are paused until the next billing cycle
- The project owner can configure a **Kaya threshold** per project in Project Settings → Configuration to control how many calls each project can use

> [!NOTE]
> Kaya AI is only accessible in projects where the owner has a Pro plan. Project owners can also control whether members have access to Kaya via the **"Members use Kaya"** toggle in Project Settings.

## Managing Your Subscription

### Accessing the Pricing Page
Navigate to the **[Pricing Page](/dashboard/pricing)** from your dashboard sidebar or from your Profile page.

### Upgrading
You can upgrade at any time. Pro-rated charges apply if you upgrade mid-billing cycle. Your new plan limits take effect immediately.

### Seat Management
Wekraft Plus and Pro are billed **per seat**. When you invite a new member to an "Admin" or "Member" role in a project you own, a new seat is automatically added to your subscription. Viewers do not count towards paid seats on the Pro plan.

### Cancellation & Downgrade
You can downgrade to the Free plan at any time:
- Premium features remain active until the end of your current billing period
- After the billing period ends, your account reverts to Free plan limits
- If you own more projects than the Free limit allows, you'll need to delete or archive projects to continue creating new ones

## Frequently Asked Questions

**Can I use the VS Code extension for free?**
Yes. The Free and Plus plans allow you to view your tasks and issues inside VS Code. However, marking tasks as complete, logging time, and AI code explanations require a Pro subscription.

**What happens if I go over my member limit?**
The system prevents you from sending new invite links or approving join requests until you upgrade or remove existing members. Existing members are not affected.

**Can I try Pro features before upgrading?**
Currently there is no free trial for Pro features. However, the Free plan includes all core features (tasks, issues, sprints, calendar, team space) with no time limit.

**Do Viewers count towards the member limit?**
Yes, Viewers count towards the total member limit per project. However, Viewers do not count towards paid seats on the Pro plan for billing purposes.

**Can different projects have different plans?**
No — your plan is tied to your account, not individual projects. All projects you own inherit your account's plan limits.
